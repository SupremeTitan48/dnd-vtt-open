# Roadmap

## Phase 1
- Core tabletop state flow with initiative, HP/conditions, and item tracking
- Session lifecycle (create, save, load) for DM hosting workflow
- Signaling API with join flow and relay-ticket support for NAT fallback architecture
- Character import and ORC/OGL-friendly content pack validation
- Starter tutorial map and walkthrough loading

## Phase 2
- Modern UI migration completion (React + Vite parity and polish)
- DM coaching flows with pacing hints and structured encounter guidance
- Expanded tutorial campaign with progressive difficulty and recap notes

## Phase 3
- Cutscene triggers and video playback hooks
- Plugin ecosystem with stable extension APIs
- Installer-grade binaries for macOS, Windows, and Linux
